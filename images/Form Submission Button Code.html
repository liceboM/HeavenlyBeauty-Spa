You can create an HTML form with a button that says "form submitted" when clicked by using HTML, CSS, and JavaScript. Here's an example of how you can do it:

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Form Button Example</title>
    <style>
        /* Style the button */
        #submitButton {
            padding: 10px 20px;
            background-color: #007bff;
            color: #fff;
            border: none;
            cursor: pointer;
        }

        /* Style the message container */
        #messageContainer {
            display: none;
            margin-top: 10px;
            font-weight: bold;
            color: #007bff;
        }
    </style>
</head>
<body>
    <form id="myForm">
        <!-- Your form fields go here -->
        <button type="button" id="submitButton">Submit</button>
    </form>

    <div id="messageContainer">Form Submitted!</div>

    <script>
        // JavaScript to show the message when the button is clicked
        const submitButton = document.getElementById('submitButton');
        const messageContainer = document.getElementById('messageContainer');

        submitButton.addEventListener('click', () => {
            // You can add form submission logic here if needed

            // Display the message
            messageContainer.style.display = 'block';
        });
    </script>
</body>
</html>

In this code:

  1. We have an HTML form with a button inside it.
  2. We use CSS to style the button and the message container. The message container is initially hidden with display: none;.
  3. JavaScript code adds an event listener to the button that listens for a click event. When the button is clicked, it displays the message by changing the display property of the message container to 'block'.

You can add your form fields and submit logic inside the JavaScript click event listener as needed. When the button is clicked, it will show the "Form Submitted!" message.
